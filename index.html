Projects

React Quote Generator w/ API

This project is my second time making use of an API and my first proper react project. It uses Math.random and Math.floor to pick a quote from those present in the API and print it to the screen on page load and also generates a new one and refreshes the quote box section when the designated button is pressed.

JavaScript Game Show App

This project was done as part of TeamTreeHouse's Front End Techdegree program. It's worth noting that the html and css were provided for me. I worked on the JavaScript exclusively. It was quite the learning experience and helped me get a better grasp on DOM manipulation.

React Markdown Previewer

One of the FreeCodeCamp JavaScript Library projects, this is a markdown previewer using React. It uses state and hooks to convert text entered into a textarea field into markdown by using the markdown library to determine what type a certain line should be and converting it accordingly.

React Drum Machine

One of the FreeCodeCamp JavaScript Library projects, this is a 'Drum Machine'. It uses an array of objects to store data points for later rendering. This project also makes use of React components to separate out the various pieces into their own files and uses import chaining to set them up so the various pieces render when needed rather than always being present.
